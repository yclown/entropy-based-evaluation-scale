## ✨ 项目简介

本项目是经典心理成长书籍《从内耗到心流：复杂时代下的熵减行动指南》中「熵型评估量表」的纯网页实现版本，无需安装、无需部署，开箱即用。

量表核心价值：通过科学的评估维度，帮使用者量化自身「心理熵值」，识别内耗根源、认知卡点与能量流失方向，最终通过熵减行动实现「从内耗到心流」的状态跃迁，适配当下复杂时代的个人心智成长需求。

本工具保留原著量表的**完整评估维度+计分逻辑+结果解读**，无二次修改核心评估体系，做到原汁原味还原书中量表的核心价值。
